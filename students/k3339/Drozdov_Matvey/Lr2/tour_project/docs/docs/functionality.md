# Функциональные возможности

## Просмотр туров
Туры отображаются списком, реализованы поиск и пагинация.

## Страница тура
- просмотр информации о туре
- бронирование (только для авторизованных пользователей)
- добавление отзывов после бронирования
- пагинация отзывов

## Мои бронирования
Пользователь может просматривать и отменять свои бронирования.

## Статистика
Реализована таблица проданных туров по странам с пагинацией.
Доступна только администратору.

## Аутентификация
Используется встроенная система аутентификации Django.
