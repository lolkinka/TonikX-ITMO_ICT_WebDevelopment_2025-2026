# Аналитика и отчёты

## Отчёт за месяц
GET `/api/analytics/monthly-report/?year=2025&month=12`

Возвращает:
- количество экземпляров книг по залам на каждый день
- количество читателей по залам на каждый день
- сколько читателей записалось за месяц по залам и всего

Параметры:
- `year` (необязательно, по умолчанию текущий)
- `month` (необязательно, по умолчанию текущий)

Пример ответа (сокращённо):
```json
{
  "period": { "year": 2025, "month": 12, "from": "2025-12-01", "to": "2025-12-31" },
  "enrollments_month": {
    "by_hall": [
      { "hall_id": 1, "hall_number": 1, "name": "Главный зал", "enroll_count": 3 }
    ],
    "total": 3
  },
  "daily": [
    {
      "date": "2025-12-01",
      "by_hall": [
        { "hall_id": 1, "books_copies": 120, "readers": 40 }
      ],
      "total": { "books_copies": 120, "readers": 40 }
    }
  ]
}
```
